## üìä L1-L4 GAP FRAMEWORK

### L1 - Critical Gaps
**Definition**: Missing information that makes content unusable or potentially dangerous

**Examples**:
- Missing safety specifications
- Incorrect technical values
- Regulatory non-compliance
- Fabricated data

**Action**: HALT - Must fix before proceeding

---

### L2 - Important Gaps
**Definition**: Missing information that significantly reduces content value

**Examples**:
- Incomplete technical explanations
- Missing context for recommendations
- Insufficient source backing
- Tier-3 source reliance

**Action**: Flag for revision before publication

---

### L3 - Minor Gaps
**Definition**: Missing "nice-to-have" information that doesn't affect core value

**Examples**:
- Additional examples
- Supplementary context
- Enhanced visuals
- Extended references

**Action**: Note for future enhancement (optional fix)

---

### L4 - Informational Notes
**Definition**: Observations, suggestions, or future considerations

**Examples**:
- Style improvements
- Alternative approaches
- Future research topics
- Related content suggestions

**Action**: Archive for reference (no immediate action)
